# 「JAKA杯」具身智能开发大赛

## 赛事主题
**你的每一行代码，都在重新定义具身智能**  
主办方：节卡机器人股份有限公司

## 赛事背景
为加速节卡具身智能开发者生态的孵化和技术沉淀，现推出分层激励计划，旨在通过阶梯式奖励机制，最大化调动开发者的参与积极性，同时高效收集不同阶段的优化反馈，推动产品迭代与生态成熟。

## 报名及投稿
### 活动对象
以JAKA具身智能产品为开发基础的开发者

### 如何参赛
1. **报名方式**  
   - 扫码报名（附二维码占位）  
   - 关注小红书 [@JAKA具身智能] 获取赛程信息[  ](https://www.xiaohongshu.com/user/profile/682fd9da0000000018020dac?xsec_token=ABqb_2v6ysUyQPKg5CN8GEVAXJ6C1KZeCS-CJYROvrfYA%3D&xsec_source=pc_search)

2. **准备作品**  
   - 包含可运行代码、测试用例、文档、项目介绍材料（PPT/视频等）

3. **提交作品**  
   - 最多提交3份作品  
   - 完成报名流程（提交信息+同意协议）  
   - 确保项目README完整（私有库需开放官方权限）  
   - Fork仓库→拷贝模板→重命名→发起Pull Request

### 投稿形式
- Demo视频：B站/小红书链接  
- 文档/源码：提交至JAKA GitHub（需开放官方可见权限）  
- 作品提交入口二维码（占位）

### 评审标准
- **性能（50%）**：数据完整度、性能指标  
- **代码质量（25%）**：规范性、接口清晰度、注释完整性  
- **创新性（25%）**：数据获取方式创新等
