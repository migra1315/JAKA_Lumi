## 报名及投稿
### 活动对象
以JAKA具身智能产品为开发基础的开发者

### 如何参赛
1. **报名方式**  
   - 扫码报名（附二维码占位）  
   - 关注小红书 [@JAKA具身智能] 获取赛程信息  
   ![](media/image1.png){width=1.7in height=1.6in}

2. **准备作品**  
   - 包含可运行代码、测试用例、文档、项目介绍材料（PPT/视频等）

3. **提交作品**  
   - 最多提交3份作品  
   - 完成报名流程（提交信息+同意协议）  
   - 确保项目README完整（私有库需开放官方权限）  
   - Fork仓库→拷贝模板→重命名→发起Pull Request

### 投稿形式
- Demo视频：B站/小红书链接  
- 文档/源码：提交至JAKA GitHub（需开放官方可见权限）  
- 作品提交入口二维码（占位）

### 评审标准
- **性能（50%）**：数据完整度、性能指标  
- **代码质量（25%）**：规范性、接口清晰度、注释完整性  
- **创新性（25%）**：数据获取方式创新等

## 加分小tips
### 💡 创新性
> **敢想敢做！**  
> - 与众不同的控制策略/场景定义（如仓库斜坡搬运球状物）  
> - 技术跨界联动（强化学习+视觉+仿真）  
> - 工具深度应用（ROS2/MuJoCo艺术化实践）  
> - 可复用、可借鉴的"生态种子"  

### ⚙️ 实用性
> **能干活才是好Demo！**  
> - 真实落地场景（仓库/实验室部署）  
> - 吃透硬件（升降柱/底盘/双臂协同）  
> - 可复现的部署流程  
> - 量化数据支撑（成功率/稳定性/能耗）  

### 🎬 演示效果
> **技术也需"出圈"！**  
> - 简明视频 > 100页PPT  
> - 清晰说明问题解决方案  
> - 适合传播的"吸睛点"（B站/小红书素材）  
> - 展现"机器为人服务"的故事感
