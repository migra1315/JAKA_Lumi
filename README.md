# 「JAKA杯」具身智能开发大赛

## 赛事主题
**你的每一行代码，都在重新定义具身智能**  
主办方：节卡机器人股份有限公司

## 赛事背景
为加速节卡具身智能开发者生态的孵化和技术沉淀，现推出分层激励计划，旨在通过阶梯式奖励机制，最大化调动开发者的参与积极性，同时高效收集不同阶段的优化反馈，推动产品迭代与生态成熟。

### 投稿形式
- Demo视频：B站/小红书链接  
- 文档/源码链接：fork JAKA GitHub+节卡小程序作品提交入口（需开放官方可见权限）  
- 作品提交入口二维码（占位）

### 评审标准
- **性能（50%）**：数据完整度、性能指标  
- **代码质量（25%）**：规范性、接口清晰度、注释完整性  
- **创新性（25%）**：数据获取方式创新等
